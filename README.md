# 2025 Week 2 - Ice Thickness Infographic (Power BI)

## Overview

This project is my solution to the **Workout Wednesday 2025 - Week 2** Power BI challenge.

The objective was to recreate an infographic that communicates the minimum recommended ice thickness required to safely support different activities using only Power BI core visuals.

---

## Challenge Requirements

- Use only Power BI core visuals
- Include a cold weather or ice-themed background
- Display each activity with both an icon and descriptive text
- Represent increasing ice thickness with increasing vertical position
- Label each ice thickness value

---

## Dataset

| Ice Thickness (Inches) | Weight Item |
|------------------------:|-------------|
| 2 | Stay Off! |
| 4 | Single person |
| 5 | Group walking in a single line |
| 6 | Snowmobile |
| 12 | Car or small truck |
| 15 | Truck or midsize SUV |
